# 🏗 System Architecture

The architecture diagram provides a high-level overview of how the application processes patient medication adherence data.

It illustrates:

- CSV-based patient data input
- Data validation
- Adherence calculation
- Risk classification
- Result generation
- Healthcare analytics visualization

---

# 🔄 Application Workflow

The workflow diagram explains the sequential processing performed by the application.

Typical workflow:

1. Load patient dataset
2. Validate input data
3. Calculate adherence percentage
4. Classify patient adherence risk
5. Generate patient summary
6. Export analysis results
7. Produce visualization charts
