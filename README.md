# AI Medication Adherence Monitor

## Overview
This project is a simple healthcare AI prototype developed using Python.

The system evaluates medication adherence rates and identifies patients at risk of missing medications.

## Features
- Medication adherence tracking
- Risk identification
- Simple healthcare analytics
- Python-based implementation

## Technologies Used
- Python
- Pandas
- CSV Dataset

## Files Included
- medication_adherence.py
- sample_patients.csv
- requirements.txt
